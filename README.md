

## Установка пакетов(из-за npm конфликты с версиями)

```bash
$ npm install --legacy-peer-deps
```

## Сборка и запуск проекта

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Запуск миграций

```bash

# Миграции
$ npx prisma migrate dev

```

## Веб управление базой данных

```bash

# Веб версия бд
$ npx prisma studio

```
